My First Pygame

Welcome to my first Pygame project! This is a simple game where you can move a box around the screen through a pop-up.
1. Click the button below to download the code for the game.
2. Next, open the downloaded file and copy the entire code.
3. Then, open a Code Editor, such as VSCode, and create a new Python file, 'main.py'.
4. Finally, paste the copied code, and run it by pressing the 'Run' button and then the 'Run Without Debugging' button.

Note: If you are using a browser that does not support downloading files directly, you can right-click the link and select "Save Link As..." to download the file.

Enjoy the game!
